Quiz Instructions

To Enjoy the Best Experience in our platform, please ensure the followings,

  1. This is an Online quiz. Please make sure that you are using the latest version of the browser. We recommend using Google Chrome
  2. It's mandatory to disable all the browser extensions and enabled Add-ons and open the test in incognito mode
  3. Once you start the test - please complete the test in one go and do not minimize the window or switch tabs
  4. If you move out of the test window/ tab more than 2 times, your test will be ended and auto-submitted
  5. You can use next and previous buttons to navigate between the problems
  6. Once you finish answering all the questions, click the finish button to submit the quiz
